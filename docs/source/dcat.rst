Socrata DCAT
============

Socrata does not support DCAT out of the box.

The following mappings are used to generate DCAT from Socrata datasets metadata.

.. list-table:: DCAT Dataset / Socrata Dataset Mappings
    :widths: 25 75
    :header-rows: 1

    * - DCAT
      - Socrata
    * - title
      - name
    * - landing_page
      - Not in metadata. Generated by combing hostname, category, name, and id
    * - license
      - Three possible entries: license.name, license.termsLink, licenseId
    * - modified
      - Avaiable sources: rowsUpdatedAt, viewLastModified
